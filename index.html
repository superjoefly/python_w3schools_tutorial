Intro Notes:

What is Python

Python is a popular programming language created by Guido van Rossum... it was released in 1991

Python Uses:

  • create web applications on a server
  • create workflows alongside software
  • connect to database systems and to read / modify files
  • handle big data and perform complex mathematics
  • induce rapid prototyping, and for production-ready software development

Reasons to Choose Python:

  • works on many different platforms (Windows, Mac, Linux etc).
  • has a simple syntax.
  • requires fewer lines of code compared to other languages.
  • code can be executed as soon as it is written...it runs on an interpreter system.
  • can be written in several ways including procedural, object-oriented and functional.

Helpful Info:

  • most recent version of Python is Python 3.
  • can be written in a text editor, as well as in an Integrated Development Environment such as Pycharm or Netbeans.

Python Syntax:

  • designed for readability.
  • uses new lines to complete a command.
  • relies on indentation and whitespace to define scope.

Getting Started

Check if python is installed (using terminal):


        python --version

Execute a python file using terminal:


        python3   "path-to-file"
        python3   hello_world.py
        exit(0)

Python Command Line

Python can be run directly in a terminal:


        python3
        print("Hello World!")
        exit(0)

Python Syntax

Python uses indentation (white-space) to indicate a block of code.


        if 5 > 2
          print("Five is greater than two!")

Python will give you an error if you skip the indentation.

The number of spaces used for indentation is user determined. However, the same number of spaces must be used within the same block of code.

Python Variables

Variables are created when a value is assigned to it:


        x = 5
        name = "Joey"

Python Comments

Python has commenting capability for the purpose of documentation:


        #This is a comment
        #This is another comment
        print("This is code that will be executed");

Comments can be used to:

  1. explain code
  2. make code more readable
  3. prevent execution of code while testing

Comments can be placed above code as well as at the end of a line.


        print("Hello World!")    #Example of end of line comment...

Here's an example of preventing code execution (commenting out):


        # print("This line won't execute!")
        print("This one will!")

To comment out multiple lines, use multiple hash tags:


        # I'm a comment
        # I'm another comment
        # I'm yet another comment!
        print("I've been commented out...")

Here is another common way to create multi-lined comments:


        """
            This just so
            happens to be
            a multi-line-comment!!!
        """
        print("Oh my!")

Variables

Variables are containers for storing data values.

Variables are created while storing values to it:


        x = 5
        print(x)

        name = "Joey"
        print(name)

Python is implicit meaning variables don't have to be declared with certain types. The type can also be changed after it has been set:


        x = 5         # x is of type integer(int)
        x = "Baby"    # x is now of type string(str)
        print(x)

Casting

To specify data type we can use type-casting:


        x = str(3)     # x will be '3'
        y = int(3)     # y will be 3
        z = float(3)   # z will be 3.0

Determine Data Type

To determine the data type of a variable, we can use the type() function:


        x = 5
        print(type(x))   # x will be of type int

        y = "Joey"
        print(type(y))   # y will be of type string

Single or Double Quotes

String variables can be declared using single or double quotes:


        name = "John"
        #   is the same as
        name = 'John'

Case Sensitivity

Variable names are case sensitive:


        name = "Joey"
        #   is not the same as
        NaMe = "Joey"

        # NaMe will not overwrite name

Variable Naming

Variables in Python can have short names (like x) or longer, more descriptive names (like name).

The following rules apply to naming variables in Python:

  • A variable name must start with a letter or the underscore character
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores
  • Variable names are case sensitive (age, Age and AGE are three different variables)

Legal variable names:


        myname = "Joey"
        my_name = "Joey"
        _my_name = "Joey"
        myName = "Joey"
        MYNAME = "Joey"
        myname2 = "Atwood"

Multi-Word Variable Names

It's important to be consistent with style and casing when creating Python programs.

The following examples demonstrate variable names that are readable:


        myFirstName = "Joey"       # Camel Case

        MyFirstName = "Joey"       # Pascal Case

        my_first_name = "Joey"     # Snake Case

        MY_VARIABLE_NAME = "Joey"  # Constants

Assigning Multiple Values to Variables

Python allows us to assign values to multiple variables in one line.

Note: the number of variables must match the number of values.


        x, y, z = "red", "orange", "yellow"

        # display the values of each variable:
        print(x)
        print(y)
        print(z)

Assigning One Value to Multiple Variables

We can assign the same value to multiple variables in one line:


        x = y = z = "bananas"

Unpacking Collections

Values of a collection (list, tuple etc) can be extracted using a process called unpacking:


        # create the array:
        fruits = ["apples", "cherries", "oranges"]

        # unpack the values into idividual variables:
        x, y, z = fruits

        # display the individual fruits:
        print(x)
        print(y)
        print(z)

Output Variables

The print() function is often used to output variables:


        # create and assign the variable:
        first_name = "Joey"

        # display the value stored in the variable:
        print(first_name)

The "+" character can be used for mathematical operations (integers) and for string concatenation:


        # Create and assign values  (integers) to the variables:
        x = 5
        y = 6

        # Add the first two variables and store in another variable:
        sum = x + y

        # Display the result:
        print(sum)

        ////////////////////

        # Create and assign values  (strings) to the variables:
        first_name = "Joey"
        last_name = "Atwood"

        # Concatenate the two string variables and store in another variable:

        full_name = "first_name" + " " + "last_name"

        # Display the result
        print(full_name)   # will display Joey Atwood

Global Variables

Global variables are variables that are created outside of a function...therefor they are within the global scope.


        # Declare / initialize a global variable:
        my_name = "Joey"

        # Define the function:
        def displayName(name):
          print("My name is " + name)

        # Call the function:
        my_name("Joey")

Local Variables

Variables created within a function are local to that function. They can only be accessed within the function.


        # Define the function:
        def display_name():

          # This is a variable that is local to the function
          my_name = "Joey"

          # Display the message:
          print("My name is " + my_name)

        # Call the function:
        display_name()

        print(my_name) # This won;t work because the variable my_name is local to the function and can only be accessed within the function.

Global Keyword

Variables created within a function are local to that function. However, we can use a the "global" keyword to create a global variable while working within a function.


        def display_name():
          print("My function is working...")

            # Declare a global variable:
            global my_name

            # Initialized the global variable with a value:
            my_name = "Joey"

        # Call the function to create the global variable:
        display_name()

        # Display the global variable:
        print("My name is " + my_name)

To change the value of a global variable while working within a function, refer to the global variable using the global keywork:


        # Create a global variable:
        my_name = "Joey"

        # Define a function which updates the global variable:
        def change_my_name():
          global my_name
          my_name = "7OEVOL"

        # Call the function which updates the global variable:

        # Display the updated global variable's value:
        print(my_name)

Python Data Types

Python variables can store data of different types. Different types of data can be used for various reasons.

Python has the following built-in data types:

Text Numbers Sequence Mapping Sets Boolean Binary
str int, float, complex list, tuple, range dict set, frozenset bool bytes, bytearray, memoryview

Retrieving Data Type

We can use the type() function to get the data type of any object:


        # declare variable:
        name = "Joey"
        age = 42

        # display the data type of the variables:
        type(name)   # str
        type(age)    # int

Python Numbers

Python has 3 numeric types:

  • int
  • float
  • complex

Variables of numeric types are created when a value is assigned to them:


        x = 1     # int
        y = 2.0   # float
        z = 1j    # complex

We can use the type() function to determine the type of an object in Python:


        print(type(x))   # Displays int
        print(type(y))   # Displays float
        print(type(z))   # Displays complex

Int

Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length:


        x = 1
        y = 8137501988374
        z = -1892340187034

        print(type(x))   # int
        print(type(y))   # int
        print(type(z))   # int
